#####  1. 验证指定ID的签章
1.  请求地址：http://xxx.com/ecs/open/verifyESealByID.jspa
2.  请求方式：POST
3.  请求参数：

| 参数名  | 类型  | 是否必需 |说明|
|:------------- |:-------------:| --------:| ---:|
|strFileData|String   |Y|待盖章的PDF文档的Base64数据|
|strESealID|String   |Y|签章ID|


4.返回结果


``` 
 {
     "meta":{
	        "code":"0",
           "message":"操作成功"
	 } 
	 "data": {
	        "签章id": false, //验证失败 
       }
	 
	 
  }
```
